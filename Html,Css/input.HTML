사용자로부터 입력받기

  • form :정보를 제출하기 위한 태그들을 포함
    autocomplete 속성: 자동완성 여부(기본 : on)
  • input :입력을 받는 요소(type 속성을 통해 다양화)
  • label :인풋 요소마다의 라벨
    for속성 값을 인풋 요소의 id와 연결. 인풋의 클릭 영역 확장
  • button : 버튼
    type속성에 submit, reset, button


    • id 속성: 페이지마다 고유해야 한다!

  • fieldset :폼 태그 내 입력요소와 라벨들을 그룹화
    disabled 속성 : 포함된 입력요소 비활성화
  • legend :필드셋 요소의 제목 또는 설명
